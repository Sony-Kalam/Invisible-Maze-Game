# Invisible-Maze-Game
A fun and challenging 3‑level invisible maze game.  
Test your memory, logic, and patience as you navigate through hidden paths!
## 🕹️ How to Play
- Find the hidden path to reach the goal.
- Each level gets trickier with longer and more complex routes.
- Don’t worry if you get stuck—trial and error is part of the fun!
## ✨ Features
- Invisible paths for extra challenge.
- Simple controls, easy to pick up.
